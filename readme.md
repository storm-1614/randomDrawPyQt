## 🛠️ 功能
随机文本按行择取文本。支持命令行参数选择名单和文件选取窗口择取文件。  
布局非常简单，为了应付当前的抽签需求。  

## 读取文件
读取的文件格式为 xlsx,格式为:
| name   | number |
|--------|--------|
| 姓名列 | 学号列 |
